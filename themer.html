Theme Controls

What does each color change?

  • Background Color: The main background of the app and input fields.
  • Card Color: The background of posts, comments, and modals.
  • Primary Color: The color of standard buttons and other interactive elements.
  • Accent Color: The main branding color, used for highlights, links on hover, and important buttons.
  • Font Color: The main text color used throughout the app.
  • Muted Font Color: The color for secondary text, like timestamps and user handles.
  • Border Color: The color for borders on cards, input fields, and dividers.
  • Link Color: The color of clickable links within post content.
